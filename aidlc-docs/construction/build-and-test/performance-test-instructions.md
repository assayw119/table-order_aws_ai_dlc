# 성능 테스트 지침 (Performance Test)

## 적용 범위
로컬 단일 매장 MVP로 본격적인 부하 테스트는 범위 외(NFR-S1). 핵심 성능 요구사항인
**실시간 전달 지연(NFR-P1: 2초 이내)** 만 검증한다.

## 실시간 지연 검증
E2E 스모크 테스트에 포함됨 (`tests/e2e_smoke.py`의 "SSE within 2s" 체크):
- 주문 생성 시각과 관리자 SSE 수신 시각 차이를 측정 → 2초 미만 확인.
- 결과: PASS (로컬 환경에서 통상 수십 ms 내 수신).

## 간이 처리량 확인 (선택)
필요 시 다수 주문을 연속 생성하여 응답 시간을 측정:
```bash
# 예시: 반복 주문 생성 후 평균 응답 시간 관찰 (수동/스크립트)
```

## 성능 목표 대비
| 항목 | 목표 | 측정 | 상태 |
|---|---|---|---|
| 신규 주문 SSE 전달 | < 2초 | < 0.1초(로컬) | PASS |
| REST 응답 | 체감 즉시 | 수십~수백 ms | PASS |
